# Runbook: Microsoft 365 Account Lockout

**Issue:** User locked out of email / Teams after too many failed login attempts.

**Symptoms:**
- "Your account is temporarily locked" message
- Can’t access Outlook or Teams

**Troubleshooting Steps:**
1. Confirm correct username (usually email address)
2. Guide user through password reset via self-service portal
3. Check for MFA prompt issues
4. Verify no recent password changes by another admin
5. Clear browser cache / try incognito mode

**Resolution:**
- Reset password and confirm MFA is set up
- Document time of lockout and number of failed attempts

**Prevention:**
Recommend using password manager and enabling MFA notifications.
