# Runbook: Wi-Fi Connection Issues (Windows 10/11)

**Issue:** User reports intermittent or no Wi-Fi connectivity.

**Symptoms:**
- Wi-Fi icon shows "No internet"
- Can see network but fails to connect
- Works on other devices

**Troubleshooting Steps:**
1. Confirm airplane mode is off and Wi-Fi is enabled
2. Forget the network and reconnect
3. Run Windows Network Troubleshooter
4. Restart router (power cycle 30 seconds)
5. Check for VPN or proxy interference
6. Update network adapter driver via Device Manager

**Common Causes:**
- Router firmware
- Outdated Wi-Fi driver
- IP conflict
- 5GHz vs 2.4GHz band issues

**Resolution Notes:**
Document signal strength, error messages, and whether issue follows the device or stays on the network.
